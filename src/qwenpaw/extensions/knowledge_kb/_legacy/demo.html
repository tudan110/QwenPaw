需要先配置启明
服务器端没检测到 DEEPSEEK_API_KEY，先配置后重启就能用了
方式 A：粘贴启明 key 到本浏览器（最快）
key 只存在你自己浏览器 localStorage，不经过服务器磁盘。切换浏览器 / 清缓存后需重设。
方式 B：服务器管理员在 zhihu 上配置（持久、多人共享）
1. 向启明平台申请一个 API key
2. 服务器终端里导入 env 并重启
export DEEPSEEK_API_KEY=sk-你的-启明-key cd /root/kb-phase2 nohup python3 server.py 38427 server.log 2>&1 &
重启后刷新本页（带 ?cb=随便数字 绕缓存），兜底按钮就是真的了。

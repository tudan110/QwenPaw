# 图表展示指南

本文档描述告警数据的图表展示方式和最佳实践。

## 图表类型

### 1. 饼图（Pie Chart）

**适用场景**：
- 告警专业分布
- 告警区域分布
- 分类占比展示

**推荐配置**：
- 半径：56%
- 颜色：使用系统默认配色
- 标签：底部居中

**示例**：
```json
{
  "title": {
    "text": "专业分布",
    "left": "center"
  },
  "tooltip": {
    "trigger": "item",
    "formatter": "{b}: {c}条 ({d}%)"
  },
  "legend": {
    "bottom": 0,
    "left": "center"
  },
  "series": [
    {
      "name": "专业分布",
      "type": "pie",
      "radius": "56%",
      "data": [
        {"name": "IPM", "value": 10},
        {"name": "TRM", "value": 5}
      ]
    }
  ]
}
```

### 2. 环形图（Donut Chart）

**适用场景**：
- 告警级别分布
- 告警状态分布
- 严重程度占比

**推荐配置**：
- 半径：["40%", "68%"]
- 颜色：使用系统默认配色
- 标签：底部居中

**示例**：
```json
{
  "title": {
    "text": "告警级别分布",
    "left": "center"
  },
  "tooltip": {
    "trigger": "item",
    "formatter": "{b}: {c}条 ({d}%)"
  },
  "legend": {
    "bottom": 0,
    "left": "center"
  },
  "series": [
    {
      "name": "告警级别分布",
      "type": "pie",
      "radius": ["40%", "68%"],
      "data": [
        {"name": "严重", "value": 5},
        {"name": "重要", "value": 8},
        {"name": "一般", "value": 4}
      ]
    }
  ]
}
```

### 3. 柱状图（Bar Chart）

**适用场景**：
- 告警标题Top
- 设备告警Top
- 数量对比展示

**推荐配置**：
- X轴：类别标签（旋转30度）
- Y轴：数量
- 柱宽：最大40
- 颜色：使用系统默认配色

**示例**：
```json
{
  "title": {
    "text": "告警标题 Top",
    "left": "center"
  },
  "tooltip": {
    "trigger": "axis",
    "axisPointer": {"type": "shadow"}
  },
  "grid": {
    "left": 48,
    "right": 24,
    "bottom": 72,
    "top": 56
  },
  "xAxis": {
    "type": "category",
    "data": ["端口DOWN", "链路中断", "CPU过高"],
    "axisLabel": {"rotate": 30}
  },
  "yAxis": {
    "type": "value",
    "name": "数量（条）"
  },
  "series": [
    {
      "name": "告警数量",
      "type": "bar",
      "barMaxWidth": 40,
      "data": [10, 5, 3]
    }
  ]
}
```

### 4. 折线图（Line Chart）

**适用场景**：
- 告警时间趋势
- 告警数量变化
- 周期性分析

**推荐配置**：
- X轴：时间
- Y轴：数量
- 平滑曲线：可选
- 颜色：使用系统默认配色

**示例**：
```json
{
  "title": {
    "text": "告警时间趋势",
    "left": "center"
  },
  "tooltip": {
    "trigger": "axis"
  },
  "xAxis": {
    "type": "category",
    "data": ["00:00", "04:00", "08:00", "12:00", "16:00", "20:00"]
  },
  "yAxis": {
    "type": "value",
    "name": "数量（条）"
  },
  "series": [
    {
      "name": "告警数量",
      "type": "line",
      "data": [5, 3, 8, 12, 10, 6],
      "smooth": true
    }
  ]
}
```

## 图表选择指南

### 按分析维度选择

| 分析维度 | 推荐图表 | 说明 |
|---------|---------|------|
| 告警级别 | 环形图 | 展示占比，中心空白更显专业 |
| 告警状态 | 环形图 | 活跃/已清除对比 |
| 告警标题 | 柱状图 | Top展示，便于对比 |
| 设备告警 | 柱状图 | Top设备对比 |
| 专业分布 | 饼图 | 分类占比 |
| 区域分布 | 饼图 | 区域占比 |
| 时间趋势 | 折线图 | 变化趋势 |

### 按数据量选择

| 数据量 | 推荐图表 | 说明 |
|--------|---------|------|
| 2-5项 | 环形图 | 清晰展示占比 |
| 5-15项 | 饼图或柱状图 | 饼图看占比，柱状图看对比 |
| 15+项 | 柱状图 | 避免饼图过于拥挤 |
| 时间序列 | 折线图 | 展示趋势 |

### 按展示目的选择

| 展示目的 | 推荐图表 | 说明 |
|---------|---------|------|
| 占比分析 | 环形图/饼图 | 直观展示部分与整体关系 |
| 对比分析 | 柱状图 | 便于数量对比 |
| 趋势分析 | 折线图 | 展示变化趋势 |
| Top展示 | 柱状图 | 降序排列，突出重点 |

## 图表配置最佳实践

### 标题

- 位置：居中
- 字体：适中大小
- 内容：简洁明确

### 颜色

- 严重告警：红色系
- 重要告警：橙色系
- 一般告警：蓝色系
- 提示告警：绿色系

### 图例

- 位置：底部居中
- 显示：关键类别
- 数量：不超过8个

### 提示框

- 触发：自动
- 格式："{b}: {c}条 ({d}%)"
- 内容：清晰有用

### 坐标轴

- X轴标签：旋转30度（避免重叠）
- Y轴单位：明确标注
- 网格线：辅助阅读

### 数据标签

- 显示：关键数据点
- 位置：合理不遮挡
- 格式：简化展示

## 图表组合

### 综合分析图表组合

1. **概览部分**：
   - 告警级别分布（环形图）
   - 告警状态分布（环形图）

2. **详细分析部分**：
   - 告警标题Top（柱状图）
   - 设备告警Top（柱状图）
   - 专业分布（饼图）
   - 区域分布（饼图）

3. **趋势分析部分**：
   - 告警时间趋势（折线图）

### 图表布局建议

- 单图表：居中显示
- 多图表：2x2网格
- 综合分析：按逻辑顺序排列

## 注意事项

### 数据准确性

- 确保数据来源可靠
- 验证数据完整性
- 处理异常值

### 图表可读性

- 避免过度装饰
- 保持简洁清晰
- 使用合适的颜色

### 性能考虑

- 大数据量时使用抽样
- 避免过于复杂的图表
- 合理设置刷新频率

### 兼容性

- 使用标准ECharts配置
- 避免使用实验性功能
- 测试多浏览器兼容

### 可访问性

- 提供文字描述
- 使用高对比度颜色
- 支持键盘导航
